# Chat Interface

The chat interface is the main component of the application, providing real-time communication capabilities.

## Features

- Real-time message updates
- Message history
- Typing indicators
- Message status (sent, delivered, read)
- File attachments support
- Responsive design
- Built-in dark-mode toggle (uses `next-themes`)

## Usage

The chat interface is implemented in `app/components/ChatInterface.tsx`. To use it in your own component:

```tsx
import ChatInterface from '@/components/ChatInterface';

export default function YourComponent() {
  return (
    <ChatInterface 
      // Add your props here
    />
  );
}
```

### Enabling Dark Mode

`ChatInterface` now includes a **Dark Mode** toggle out of the box. For the toggle to work you must:

1. Wrap your application (e.g. in `app/layout.tsx`) with `ThemeProvider` from **next-themes**:

```tsx
import { ThemeProvider } from 'next-themes';

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="en">
      <body>
        <ThemeProvider attribute="class" defaultTheme="system" enableSystem>
          {children}
        </ThemeProvider>
      </body>
    </html>
  );
}
```

2. Ensure Tailwind is configured for class-based dark mode (already set in `tailwind.config.js`):

```js
module.exports = {
  darkMode: 'class',
  // …
};
```

Once these prerequisites are met, users can switch between light and dark themes directly within the chat header.

## Customization

The chat interface can be customized through props and CSS styling, including Tailwind’s dark-mode utilities. See the component documentation for available options.
