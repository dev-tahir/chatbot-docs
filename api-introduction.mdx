# API Documentation

The Chatbot API provides a set of endpoints for managing chat functionality, user authentication, administrative tasks, and blog content.

## Base URL

```
https://your-domain.com/api
```

## Authentication

Most API endpoints require authentication. Include the authentication token in the Authorization header:

```bash
Authorization: Bearer YOUR_ACCESS_TOKEN
```

## API Endpoints

The API is organized into the following categories:

- `/api/auth/*` - Authentication endpoints (NextAuth.js)
- `/api/chat/*` - Chat functionality
- `/api/blog/*` - Blog posts and related content
- `/api/users/*` - User management
- `/api/users/stats/*` - User statistics

For detailed information about specific endpoints, see their respective documentation pages.
